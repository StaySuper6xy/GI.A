# Gia - Business Management Solution

An AI-integrated business management solution built with:
- Frontend: Next.js, React, TypeScript
- Backend: Node.js, Express, MongoDB

## Setup

### Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

### Backend
\`\`\`bash
cd backend
npm install
npm run dev
\`\`\`
